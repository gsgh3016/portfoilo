# 반응형 동작 규칙

## R6. 반응형 동작

- **R6.1.** 화면 너비가 변경되어도 컬럼 너비(150px)는 유지되어야 한다.
- **R6.2.** 화면 너비가 줄어들면 컬럼 수만 감소하고, 아이템은 기존 배치를 유지하려고 시도한다.
- **R6.3.** 아이템이 그리드 영역을 벗어나면 **에러가 발생**해야 한다. (런타임 에러 또는 개발 시점 검증 에러)
