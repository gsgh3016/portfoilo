# 컴포넌트 구조 규칙

## R8. 사용자 편집

- **R8.1.** 사용자가 드래그 앤 드롭 등으로 레이아웃을 편집하는 기능은 필요하지 않다.
- **R8.2.** 레이아웃은 개발 시점에 고정 배치된다.

## R9. 컴포넌트 구조

- **R9.1.** 그리드 컨테이너는 재사용 가능한 React 컴포넌트로 구현되어야 한다.
- **R9.2.** 그리드 아이템은 자식 컴포넌트로 주입 가능해야 한다.
- **R9.3.** 아이템의 위치와 크기 정보는 props 또는 데이터 구조로 전달되어야 한다. (배치 정보 저장 방식은 [../design-options/](../design-options/) 참조)
