# 검증 규칙

## R7. 아이템 겹침 방지

- **R7.1.** 아이템 간 겹침은 허용되지 않는다.
- **R7.2.** 겹침이 감지되면 에러가 발생하거나 경고가 표시되어야 한다.
- **R7.3.** 겹침 검증은 개발 시점 또는 런타임 초기화 시점에 수행된다.
