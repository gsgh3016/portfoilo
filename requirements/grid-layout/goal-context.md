# 그리드 레이아웃 시스템 목표 및 컨텍스트

## Goal

개인 블로그 웹사이트의 대시보드 페이지를 위한 고정폭/고정높이 그리드 레이아웃 시스템을 구축한다. 벽걸이 네트 망처럼 전체 화면을 그리드로 나누어, 다양한 크기의 컴포넌트를 수동으로 배치할 수 있도록 한다. 모든 환경(모바일, 태블릿, PC)에서 컬럼 너비는 절대적으로 동일하게 유지된다.

---

## Context

- **프로젝트**: 개인 블로그 웹사이트 (포트폴리오)
- **기술 스택**: Next.js (App Router) + React + TypeScript
- **사용 목적**: 대시보드 형태의 메인 페이지 레이아웃
- **레이아웃 스타일**: 벽걸이 네트 망 형태의 고정 그리드 시스템
