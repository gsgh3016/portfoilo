# 그리드 레이아웃 시스템 가정

## Assumptions (확정된 가정)

- **A1.** 행 높이는 **150px**로 확정 (컬럼 너비와 동일)
- **A2.** 그리드 간격(gap)은 **10px**로 확정
- **A3.** 아이템 겹침은 **허용하지 않음**. 감지 시 에러 발생
- **A4.** 컬럼 수 계산 시 소수점은 내림(floor) 처리. (1920px ÷ 150px = 12.8 → 12개)
- **A5.** 아이템이 그리드 영역을 벗어나면 **에러 발생** (스크롤, 숨김, 재배치 없음)
- **A6.** 사용자 편집 기능은 **불필요**. 개발 시점에 고정 배치
- **A7.** 배치 정보 저장 방식은 **옵션 2 (배열 데이터 구조)** 권장
